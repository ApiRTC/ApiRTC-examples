ApiRTC Starting Guide


Create an account

First thing to do is to create an account to get your apiRTC key

Customize your userId

Then you can customize your client Id. For instance, if you have a database on your application you can use the userId that is defined in this database
You only have to add the parameter "apiCCId" in the apirtc.init() function in set it with your userId.
For example, to set a userId to "1" :
apiRTC.init({
apiKey : "#INSERT_YOUR_API_KEY_HERE#", // --> Get your API Key at https://cloud.apirtc.com/enterprise/api (free account required)
apiCCId : "1",
onReady : sessionReadyHandler,
ccsServer : "ccs3.apizee.com"
});
