Tutorial 7: ApiRTC client with recording feature

This sample is accessible on Github


This sample shows an apiRTC client that enable calls recording.

At the end of the call an apiRTC event is fired that enables you to get all the needed informations to play the recorded files.

Check the comment "Record" in the source code to know what need to be added on your apiRTC application for the recording feature.

Used apiRTC functions are :

  • - webRTCClient.call() with option 'RECORD'
  • - webRTCClient.setRecordedCall() can be use to set an option to RECORD all the calls (this is not shown in this demo)

Used apiRTC events are :

  • - recordedStreamsAvailable

Replay : It is now possible to play the file that is being recorded during the call : use the button "Replay" to open our tutorial Replay page to access the stream
