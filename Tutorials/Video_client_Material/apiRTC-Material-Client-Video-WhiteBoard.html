Tutorial 13 : Whiteboard

This sample is accessible on Github


This sample shows an apiRTC client build with Material design.
With this demo you can:
  • o Establish video calls with different clients.
  • o Start a WhiteBoard session with another user
  • o Start a WhiteBoard session offline

Open another client
Your browser needs to support canvas for this to work!

Start Drawing !

Enter the userId of the user you want to join, then start the whiteboard by creating a room : "createRoom". it will invite your contact

You will then be able to work together on your drawing
